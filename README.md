# Advanced RAG Course 2026

A 77-page technical course that takes engineers who have built a basic RAG pipeline 
and brings them to top-decile practitioner level.

## What's covered

- **Module 1** — Systems view, failure taxonomy, RAG vs fine-tuning vs CAG
- **Module 2** — Embeddings deep dive: MRL, contrastive loss, 2026 model landscape
- **Module 3** — Indexing: contextual retrieval, late chunking, RAPTOR, document parsing
- **Module 4** — Vector stores: HNSW, IVF, filter strategies, DB selection
- **Module 5** — Retrieval strategies: hybrid search, RRF, ColBERT, HyDE, reranking cascades
- **Module 6** — Generation: lost-in-the-middle, context assembly, Self-RAG, CoVe
- **Module 7** — Evaluation: NDCG, faithfulness, LLM-as-judge, RAGAS, G-Eval
- **Module 8** — Advanced architectures: GraphRAG, ColPali, Adaptive RAG, CAG
- **Module 9** — Agentic RAG: ReAct, Reflexion, multi-agent, tools as retrieval
- **Module 10** — Production: latency budgets, caching tiers, observability, prompt injection defense
- **Module 11** — Capstone: reference architecture, debugging protocol, readiness checklist
